## 그래프 순회 (Graph Traversal)
그래프 순회란 그래프 탐색(search)라고도 불리우며 그래프의 각 정점을 방문하는 과정을 말한다.
그래프 순회에는 크게 깊이 우선 탐색(Depth-First Search)과 너비 우선 탐색(Breadth-First Search)의 2가지 알고리즘이 있다.

- 모든 정점을 다 방문해야 한다.

### BFS
- queue를 이용해서 문제를 푼다.
- 시작 노드에서 가까운 곳부터 탐방해서 문제를 푼다.

### DFS
- stack을 이용해서 문제를 푼다.
- 재귀 함수를 사용한다.
